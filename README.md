#Dmsg(分布式实时通讯框架)
##概述
应对单机消息路由处理能力薄弱，提升实时消息准确可靠的能力，分布式处理实时消息。提供一套消息处理框架，包括消息的过滤，鉴权，缓存接口，让三方开发者可以快速的搭建自己的分布式消息处理框架。
